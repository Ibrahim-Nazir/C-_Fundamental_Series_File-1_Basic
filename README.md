# C-_Fundamental_Series_File-1_Basic
In this file, I practiced C++ fundamentals as a beginner when I first started learning the language. I solved four programs to build my foundational skills:

1- Check Leap Year: A program to determine if a specific year is a leap year.

2- Assign Student Grade: A program that assigns a letter grade based on a student's marks.

3- Calculate Student Percentage and Show Status: A program to calculate a student's percentage and display their pass/fail status.

4- Press Key to Show Month Name: A program that displays the name of a month based on a numeric key input (e.g., 1 for January).
